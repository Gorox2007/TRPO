# Telegram Dating Bot - Stage 1

Этот репозиторий содержит **Этап 1: Планирование и проектирование** Telegram-бота знакомств.

## Что уже сделано

1. Описаны компоненты учебного MVP: [docs/01-services.md](docs/01-services.md)
2. Спроектирована архитектура и схема взаимодействий: [docs/02-architecture.md](docs/02-architecture.md)
3. Подготовлена схема БД и стартовый SQL: [docs/03-database-schema.md](docs/03-database-schema.md)

## Цель продукта

Подбор кандидатов на знакомства на основе:
- анкеты пользователя (возраст, пол, интересы, гео),
- полноты анкеты,
- количества фотографий,
- первичных предпочтений (возрастной диапазон, пол, город).

## Архитектура MVP

Один монолит:
- `Bot` (Telegram обработчики),
- `Core` (бизнес-логика),
- `PostgreSQL` (данные).

## Следующие этапы (черновик)

1. Реализация MVP-бота и команд (`/start`, анкета, предпочтения)
2. Реализация логики скоринга и выдачи кандидатов
3. Реализация взаимных лайков и чата/интеграции с Telegram
4. Модерация, аналитика, A/B тесты
