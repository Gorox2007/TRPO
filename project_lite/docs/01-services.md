# 1. Компоненты учебного MVP
## 1. Bot (Telegram слой)

**Назначение:** принимает события из Telegram и отправляет ответы пользователю.

Основные функции:
- команды `/start`, `/profile`, `/profile_set`, `/profile_delete`, `/prefs`, `/next`, `/like`, `/skip`;
- валидация входных данных;
- вызов внутренней логики `Core`.

## 2. Core (вся бизнес-логика)

**Назначение:** хранит и выполняет основную логику приложения.

Основные функции:
- анкета: возраст, пол, интересы, город/гео;
- полнота анкеты (`profile_completeness`);
- фото и проверка минимального количества;
- предпочтения: диапазон возраста, пол, город;
- подбор кандидатов и расчет итогового score;
- действия пользователя (`like/skip/block`) и создание `match`.
- кэширование предварительно отранжированной очереди кандидатов.

## 3. Data (PostgreSQL)

**Назначение:** постоянное хранение данных.

Хранит:
- пользователей и анкеты;
- фото;
- предпочтения;
- действия и совпадения (матчи).
